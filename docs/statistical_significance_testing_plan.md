# 模型性能比较：显著性检验与效应量计算方案（草案）

## 1. 目标与比较对象
对模型准确率（Accuracy, %）进行**配对统计比较**、多重比较校正、效应量计算，并**更新/补全 95% 置信区间（CI）**。

计划执行的两组比较（同一评估单元上的配对结果）：
1. Lite-AMSNet vs Lite-AMSNet w/MSPA  
2. Lite-AMSNet vs InceptionTime

已给定汇总指标（待核对其统计含义）：
- Lite-AMSNet：98.20% ± 0.08%（需确认“0.08%”是 SD/SEM/CI 半宽）
- Lite-AMSNet w/MSPA：98.04%
- InceptionTime：97.91%

## 2. 数据与前置确认（配对检验的必要条件）
**配对 t 检验需要逐个对应的观测值**（同一折/同一被试/同一随机种子/同一重复实验）。

最低数据需求（二选一）：
- **优先**：每个评估单元的准确率序列  
  - `acc_lite[i]`, `acc_mspa[i]`, `acc_inception[i]`, `i=1..n`
- **备选（不推荐）**：在无法提供原始配对序列时，需要额外提供能恢复“差值方差”的信息  
  - `n`（折数/被试数/重复次数）  
  - 各模型准确率的 SD（非 SEM）  
  - 以及配对相关/差值 SD（否则无法做严格的配对 t 检验与 d_z）

同时需要确认：
- 这些准确率是 **LOSO 的每被试结果**、还是 **k 折交叉验证每折结果**、还是 **多随机种子重复**。
- “98.20% ± 0.08%”中的 `0.08%` 表示：`SD` / `SEM` / `95% CI 半宽` / 其他。

## 3. 统计检验：配对 t 检验（两侧）
对每个比较构造差值向量（按评估单元逐一配对）：
- 比较 1：`d1[i] = acc_lite[i] - acc_mspa[i]`
- 比较 2：`d2[i] = acc_lite[i] - acc_inception[i]`

对每个差值向量执行**两侧配对 t 检验**：
- `t = mean(d) / (sd(d) / sqrt(n))`
- 自由度：`df = n - 1`
- 输出：`t`, `df`, 原始 `p` 值

解释方向：
- `mean(d) > 0` 表示 Lite-AMSNet 平均准确率更高；`< 0` 则相反。

## 4. 多重比较：Bonferroni 校正
本次属于 2 个预先指定比较（family size `m=2`）：
- 校正显著性水平：`alpha_adj = 0.05 / 2 = 0.025`
- 或等价报告校正后的 p 值：`p_adj = min(p * 2, 1)`

判定规则（推荐统一使用校正后 p 值）：
- 若 `p_adj < 0.05` 则认为在 Bonferroni 校正后仍显著。

## 5. 效应量：Cohen’s d（配对设计）
使用配对样本常用的效应量 **Cohen’s d_z**：
- `d_z = mean(d) / sd(d)`

报告内容：
- `d_z` 数值与符号（正/负对应差值方向）
- 可选：为 `d_z` 给出 95% CI（推荐使用对配对单元的 bootstrap 重采样；或非中心 t 方法）

解释参考（仅作经验尺度，不替代领域判断）：
- 0.2（小）、0.5（中）、0.8（大）

## 6. 95% 置信区间（CI）更新方法
### 6.1 各模型平均准确率的 95% CI（基于 t 分布）
对每个模型的 `acc[i]`：
- 均值：`mean(acc)`
- 标准误：`SE = sd(acc) / sqrt(n)`
- `CI95 = mean(acc) ± t_{n-1,0.975} * SE`

说明：
- 若给定的“±”是 SD/SEM/CI 半宽不同含义，CI 的计算方式会不同；因此必须先确认原始定义或直接使用原始 `acc[i]` 重算。

### 6.2 两模型“平均差值”的 95% CI（推荐与配对 t 检验一起报告）
对 `d[i]`：
- `CI95(diff) = mean(d) ± t_{n-1,0.975} * sd(d)/sqrt(n)`

解释：
- 若 `CI95(diff)` 不包含 0，则与两侧配对 t 检验结果一致（未校正情况下）。

## 7. 质量检查与假设前提
配对 t 检验主要假设：差值 `d[i]` 近似正态。计划检查：
- `d[i]` 的分布（直方图/QQ 图），以及极端离群点
- 若明显偏离正态且 `n` 较小，可补充（作为敏感性分析）：
  - Wilcoxon signed-rank（配对非参数检验）
  - 或对差值做 bootstrap CI（更稳健）
（是否执行敏感性分析以你的需求为准）

## 8. 最终交付物（输出格式）
1. 各模型：`mean accuracy` + `95% CI`
2. 每个比较：`mean(diff)` + `95% CI(diff)`、`t(df)`、`p`、`p_adj`（Bonferroni）、`Cohen’s d_z`（可选 `CI`）
3. 简短结论：在 Bonferroni 校正后是否显著、效应量大小与方向
